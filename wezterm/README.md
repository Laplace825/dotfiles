# KeyMap for wezterm

`Command` -> `Cmd`
`Shift` -> `Sh`
`Option` -> `Op`

| key map | function | 
| :------------------------: | :----------------------------: |
| \<Cmd\> + T | open a new tab  | 
| \<Cmd\> + W | close a new tab  | 
| \<Cmd\> + R | reload configuration  | 
| \<Op\> + Enter | full screen  | 
| \<Cmd + Sh\> + [ or \] | change between the tabs  | 
| \<Cmd\> + number(1 2 3...) | change between the tabs  | 
| \<Cmd\> + `+` or `-` | change the current font size   |
